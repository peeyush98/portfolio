Electric Charging Stations

Analyzed the number of EV charging stations at the zip code level for the United States using IRS data. A classification model was used to predict if a zip code had charging stations within it and a regression model was used to predict the number of charging stations. The analysis helps the charging station infrastructure developers prioritize zip codes.
